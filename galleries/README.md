## Images that are in the photo Gallery

The metadata.yml file needs to be edited after changing photo images.

The metadata.yml file currently is like the following.
Note that the three hyphens "---" between information about each image must be kept.
The filenames do not have to include any numbering. E.g. tesla_1.jpg could be named tesla_tower.jpg
The caption can not start with a number. E.g. Can not have: caption: 1. The Big Tower. 

```yaml
---
name: green.png
caption: No.1. Test Green Landscape 1920 x 1080.
---
name: pink.png
caption: No.2. Test Pink Landscape 1920 x 1080.
---
name: teal.png
caption: No.3. Test Teal Landscape 1920 x 1080.
---
name: teal_p.png
caption: No.4. Teal Portrait 1080 x 1920.
---
name: blue.png
caption: No.5. Blue Landscape 1024 x 768.
---
name: dark-green-p.png
caption: No.6. Dark-green Portrait 768 x 1024.
---
name: dark-red.png
caption: No.7. Dark-red Landscape 800 x 600.
---
name: light-green-p.png
caption: No.8. Light-green Portrait 600 x 800.
---
name: purple.png
caption: No.9. Purple Landscape 640 x 480.
---
name: purple-p.png
caption: No.10. Purple Portrait 480 x 640.
---
```
